# Health Dashboard 

A personal habit tracking dashboard built with Python and Streamlit.

## Features
- Track daily sleep hours and workout habits
- Rolling 7-day average sleep chart
- KPI metrics (total days, average sleep, workout days)
- Upsert logic — edit past entries without duplicates
- Download data as CSV

## Tech Stack
- Python, Streamlit, SQLite, pandas, matplotlib

## How to Run
pip install streamlit pandas matplotlib
streamlit run app.py
